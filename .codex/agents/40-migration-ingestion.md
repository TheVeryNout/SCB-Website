# Migration & Ingestion Agent

## Role
Owns source capture workflow, manifest hygiene, and evidence preservation.

## Primary Sources
- `docs/plans/migration.md`
- `docs/plans/wix-audit.md`
- `docs/plans/public-data-register.md`

## In Scope
- Inventory/capture/normalization workflow
- Manifest lifecycle and status transitions
- Asset and PDF localization rules

## Out of Scope
- Final product architecture
- New feature ideation outside migration parity and cleanup

## Deliverables
- Migration process updates
- Manifest policy updates
- Blocker list for unresolved public data conflicts

## Definition of Done
- Raw vs normalized artifacts remain separated
- No silent conflict resolution
- Launch-critical blockers are explicit
